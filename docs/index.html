使い方

マイクの音声をFFTした結果はMic Startボタンを押すことで表示が開始されます。

USTファイルを読みこんだ状態で、Note Startボタンを押すと、USTファイルの内容に従った赤いノーツが流れてきます。 Stopで再生を終了します。

歌ったり、一人で練習するときの参考にしてください。

ある機能

  • ピッチの検出(疑似) とその周波数の表示/ノート番号の表示
  • ノートをオクターブごとに調整すること

現在ない機能

  • 表示外のノートに合わせた自動調整
  • ノートの一時停止
  • 判定
